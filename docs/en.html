Why it's handy

Built around one thing — watching while you work — with every little annoyance solved.

⚡

Tiny & install-free

A single executable, double-click and go. Shares the system browser engine instead of bundling a whole Chromium, so it's light on RAM and stays smooth in the background.

📌

Always on top

Pin it to the bottom-right corner, above Excel, your IDE or browser — no more constantly switching windows to find your video.

🕶️

Boss key

Press Ctrl+Shift+Z and the window vanishes instantly while the video pauses (audio stops too — no giveaway). Press again when the coast is clear.

🎬

Video-only

One click hides the title, recommendations and comments; the player fills the whole window. Works for normal YouTube videos and Shorts.

🚀

Service shortcuts

Jump to YouTube / Netflix / Bilibili in one click, and "open in new tab" links are intercepted to open in the same window (fixes Bilibili videos not opening).

🪟

Drag & resize freely

Frameless design — drag the red handle to move, edges/corners to resize. The address bar auto-hides for a clean view.

Download

Portable build recommended on Windows — download and double-click.

🪟 Windows portable (no install) 🪟 Windows installer (.exe / .msi) 🍎 macOS (.dmg) 🐧 Linux (.AppImage / .deb / .rpm) The portable build just needs a double-click. WebView2 Runtime is built into Windows 11; on Windows 10 install it from Microsoft if missing.

Get started in 3 steps

No setup — open it and go.

1
Download your platform's file from Releases; on Windows just double-click to run.
2
The window opens bottom-right, always on top. Drag the red handle to move, edges/corners to resize; press Ctrl+L to open the address bar and switch videos.
3
Boss walking by? Press Ctrl+Shift+Z to instantly hide and pause; press again to bring it back.

🔒 Security & transparency

The files you download are auto-built from public source code — relatively safe.

Every release binary is built automatically by GitHub Actions directly from this repo's open-source code — nobody hand-packages builds from their own computer. The source and the full log of every build are public and auditable, so the file you download corresponds to the public code and you can be relatively assured it carries no bundled virus or backdoor.

Note: automated builds greatly reduce the risk of sneaked-in malware, but can't 100% guarantee absolute safety. If you'd rather be sure, clone the source and build it yourself with npm run tauri build.
